============================================
How to use 0-lookup.py
============================================

This library is very simple, since it only has one function called
 ''lookup(obj)''.

Numbers:
========
::With examples in the main.py file

>>> lookup = __import__('0-lookup').lookup
>>> class MyClass1(object):
...    pass
>>> class MyClass2(object):
...    my_attr1 = 3
>>> def my_meth(self):
...        pass
>>> print(lookup(MyClass1))

>>> print(lookup(MyClass2))
['__class__', '__delattr__', '__dict__', '__dir__', '__doc__', '__eq__', '__format__', '__ge__', '__getattribute__', '__gt__', '__hash__', '__init__', '__init_subclass__', '__le__', '__lt__', '__module__', '__ne__', '__new__', '__reduce__', '__reduce_ex__', '__repr__', '__setattr__', '__sizeof__', '__str__', '__subclasshook__', '__weakref__', 'my_attr1']
